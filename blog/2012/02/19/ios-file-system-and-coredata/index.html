IOS File System and CoreData

Feb 19th, 2012

此篇文字用于记录学习IOS文件系统与CoreData的总结与笔记，以作备忘！

IOS把每个内部的文件系统是基于Unix的，不过做了文件保护，它把每个应用都放在一个独立的沙盒（sandbox）里面，应用只能访问操作自己沙盒里的文件。

文件系统

这个沙盒的目录包括应用程序包（Application bundle）目录，文档目录，缓存目录等目录组成。应用包目录是不可写的，文档目录类似MAC OS上的My Document，是操作文件的主要场所。其他的目录可以用　NSSearchPathDirectory　查找。
